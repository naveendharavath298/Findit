# 🔍 FindIt — Lost & Found Platform

FindIt is a full-stack web application that helps users report, track, and recover lost items through a smart matching system, secure authentication, and real-time communication.

🌐 **Live Demo:** https://findit-rieg.onrender.com  

---

# 🚀 Features

## 🔐 Authentication
- Login with Email & Password
- OTP-based login (Email / Phone)
- JWT-based session management

## 📦 Lost & Found System
- Report lost items
- Report found items
- Categorized listings (Electronics, Documents, Accessories, etc.)
- Search & filter items

## 🤖 Smart Matching System
- Automatically matches lost & found items
- Uses:
  - Category similarity
  - Text similarity
  - Location matching
- Notifies users when a match is found

## 💬 Chat System
- Secure chat between matched users
- Enabled only after verification
- Real-time messaging (API-based)

## 🔔 Notifications
- Match alerts
- System notifications
- Read/unread tracking

## 📊 Dashboard & Stats
- View your reports
- Track matches
- See platform statistics
